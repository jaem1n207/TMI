# ts-movies

> 해당 링크를 통하여 이용해 보실 수 있습니다. [TMI](https://jaem1n207.github.io/TMI/ "Go TMI")

> API used in project: [The Movie Database API](https://www.themoviedb.org/ "Go TMDB")

## Introduce
### Movie
* Top Rate Movie
* Movie Trailer
* Movie Detail
* Popular Movie
* Upcoming Movie
* NowPlaying Movie
* Recommendation Movie
* Cast Info
* Similar Movie
* Detail Cast

### Project Stack
* React
* React Router
* TypeScript
* Redux
* Redux Thunk
* Container Presenter Pattern
* Scss
* React Helmet
* Axios
* Immer
* Redux-logger
